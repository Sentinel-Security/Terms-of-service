Términos de Servicio de Sentinel

Última actualización: 03/10/2025

Al invitar y utilizar el bot Sentinel en tu servidor de Discord, aceptas los siguientes términos:

1. Recolección y uso de datos

Sentinel puede almacenar los siguientes datos con el fin de ofrecer correctamente sus funciones:

  • Logs de comandos utilizados
  • IDs de usuarios
  • Nombre del servidor
  • Invitación directa al servidor (si es proporcionada para funciones internas)

Estos datos no se comparten con terceros y se usan únicamente para el funcionamiento y registro interno del bot.
Puedes solicitar la eliminación de los datos almacenados de tu servidor o usuario contactando al soporte.

2. Uso permitido

Está estrictamente prohibido usar Sentinel para:

  • Spam, raids o automatizaciones malintencionadas
  • Acoso, insultos o actividades discriminatorias
  • Violaciones a los Términos de Servicio de Discord

Sentinel puede ser expulsado automáticamente de servidores que hagan mal uso del bot.

3. Disponibilidad

Aunque se intentará mantener el bot activo en todo momento, no se garantiza un funcionamiento continuo.
Sentinel puede dejar de funcionar temporal o permanentemente sin previo aviso.

4. Contacto y soporte

Para dudas, sugerencias o solicitudes de eliminación de datos:

Servidor de soporte: https://discord.gg/yF5XkcTPEq
